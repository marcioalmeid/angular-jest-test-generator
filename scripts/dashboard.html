Instalar Localmente

Linkar a Projeto
Cria link do gerador em um projeto Angular específico.
./scripts/link-to-project.sh [projeto]

Criar Tarball
Cria arquivo .tgz para distribuição manual ou compartilhamento.
./scripts/create-tarball.sh

Configura Jest completo em um projeto Angular com todos os arquivos necessários.
./scripts/setup-project.sh [projeto]
